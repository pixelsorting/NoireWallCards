# NoireWallCards
Work in progress ROUNDS mod, adding elemental and Binding of Isaac inspired cards

## Cards:
 - Thunderstruck: 
	 - Bullet impacts now call forth the wrath of Zeus
	 - +10% Damage
	 - -35% Attack Speed
 - Continuum:
	 - Your bullets wrap around the screen but have decreased lifetime
	 - -65% Bullet Gravity
	 - -45% Bullet Speed
	 - 5s Bullet Lifetime
